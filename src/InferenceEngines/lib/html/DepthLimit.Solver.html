A basic backtracking solver with a depth limit, hence no risk of entering an infinite search.

Nota. When the limit is set to a very high value, it behaves essentially like the na√Øve version. Conversely, when the limit is set too low, it may miss solutions. Determining the right value is problem-dependent, based either on empirical knowledge or known mathematical properties.

Post-condition.

  • For any solution, the sequence of transformations is indeed a solution.
  • No solution can be of length greater than the limit.
  • Returns a list of solutions, each one being a list of transformations to apply in sequence from the initial state in order to reach a final state, i.e., solve the problem at hand
