The DepthLimit has been introduced in order to avoid entering an infinite search in an infinitely deep search space. However, using it directly is hard since we have to provide a maximal depth. For instance, it can find lengthy solution before much shorter ones because it entered some sub-space before a more promising one. By using it through "Solver.MetaIteratedDeepening" (Meta), we avoid this problem; the shorter solutions will be found sooner. This solver is often named IDS for Iterated Deepening (Best-first) Search. Its graph-based counterpart is the /breath-first/ search algorithm.
