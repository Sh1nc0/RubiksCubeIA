Various Functions Used in this Library

This module groups together a number of functions that are not present in standard libraries but used as utilities in various parts of the package.

  • Author(s): : (c) February 28, 2019, José Martinez, École polytechnique de l'université de Nantes
val consecutives : 'a list -> ('a * 'a) list

A function that returns the pairs of consecutive elements in a list.

Example:

consecutives "hello" returns [('h','e'),('e','l'),('l','l'),('l','o')]

Post-condition:

  • The tail of the list can be rebuilt from the second components of the pairs.
  • Unless the list contains a single element, it can be rebuilt from the pairs.

Time complexity

O(n) where n is the number of elements in the list.

  • Returns the list of pairs of consecutive elements of the parameter.
Parameters:
xs :
'a list
 
val splits : 'a list -> ('a list * 'a list) list

A function that mimics part of the Prolog "append" predicate. It splits a list at every index, returning a list of pairs of sub-lists.

Example:

splits "hello" returns [("","hello"),("h","ello"),("he","llo"),("hel","lo"),("hell","o"),("hello","")]

Post-condition:

  • The concatenation of each pair gives back the original list.

Time complexity:

O(n) where n is the number of elements in the list.

  • Returns the decompositions of xs into two sub-lists, including the case where one of them is empty.
Parameters:
xs :
'a list
a list of elements 
val splits3 : 'a list -> ('a list * 'a list * 'a list) list

An extension of 'splits' that splits a list into three consecutive sub-lists.

Example: splits "hello" returns a lengthy [("","","hello");("","h","ello");("","he","llo");("","hel","lo");("","hell","o");("","hello","");("h","","ello");("h","e","llo");("h","el","lo");("h","ell","o");("h","ello","");("he","","llo");("he","l","lo");("he","ll","o");("he","llo","");("hel","","lo");("hel","l","o");("hel","lo","");("hell","","o");("hell","o","")]

Post-condition:

  • The concatenation of the sub-lists gives back the original list.
  • Returns he decompositions of xs into three consecutive, possibly empty, sub-lists. Note that in the above example ("hello","","") is not in the result but in practice we do not need it.
Parameters:
xs :
'a list
a list of elements

val kronecker : bool -> int

The Kronecker operator simply translates a boolean into 0 or 1.

Example:

kronecker (allDifferent "hello") returns 0

  • Returns the translation of the argument as a natural integer, respectively 0 or 1.

Cartesian product

val cartesianProduct : 'a list -> 'b list -> ('a * 'b) BatSet.t

Set of all the couples that can be built from two sets. We use pervasive polymorphic comparaison (<).

We use BatSet.t to represent sets, form the library "OCaml Batteries Included" instead of the module Set of the standard library because it is easier to use when we don't need specific comparison operations.

Nota Since we are actually working with lists, there can be duplicates in them.

  • Returns a set containing all the couples consisting of an element of xs and an element of ys.
Parameters:
xs :
'a list
: a collection of some elements

'b list
: a collection of other elements
